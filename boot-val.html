验证框架测试v1.0.0

koala.validate是一个通用的验证框架，完成异步的前台以及后台的数据验证。 可扩展，可定制，灵活方便，能够节省大量验证编码工作

一般的页面验证的方法，启动框架预设的验证类型：

如下所示，给控件添加 required 属性以进行必需的验证 

<script>
/* 全局参数设定, 启动验证框架 */
$.validate();
</script>
<body>
	<!-- 为文本框添加required验证 -->
	<input required>
</body>

也可以定制和添加自定义的验证类型

<script>
/* 全局参数设定, 添加自定义的验证, 并启动验证框架 */
$.validate({
	validators : {
		/*添加文本框的输入规则,不能输入'admin'*/
		forbid : {
			target : "#text",
			type : "forbid",
			message : "这个词禁止输入",
			handler : function(value){
				return value != "admin";
			}
		}
	}
});
</script>
<body>
	<input id="text" required>
</body>
